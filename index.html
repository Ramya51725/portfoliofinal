Static Website Challenge

This code uses CSS Grid to arrange the boxes. The class .hcf is the main grid container. It has 9 columns and 6 rows. The gap gives space between the boxes. Each box uses grid-column and grid-row to choose its place in the grid. For example, grid-column 1/3 means the box starts at column 1 and ends at column 3. This makes the box bigger. By changing the column and row numbers, I can make the boxes big or small and place them in different positions..

The initWarmup() function is used to load warm-up exercises from the backend. It first displays the warm-up overlay and determines the correct level based on the user’s current month. Then it retrieves the category ID from local storage and sends a request to the API to fetch exercises for that level. If exercises are available, it shuffles them, selects six exercises, and displays the first one. If no exercises are found or an error occurs, the overlay is hidden.
